## ✨ Features

- 🔘 Circular rating buttons with selection state
- 🧠 Stores selected rating using URL parameters
- 🔄 Smooth navigation to thank-you screen
- 📱 Fully responsive for mobile and desktop
- 💅 Clean, modern glassmorphism design
